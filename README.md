# Videostreamer 


Short description:
A simple streaming network

# Technologies used:
- React
- JSON-server
- node-media server
- flv.js


To run it you should:

# Running Locally
1. git clone https://...
2. cd API
3. npm install
4. cd Client
5. npm install
6. cd rtmpserver
7. npm install
8. npm start

# Dependecies :
- json-server
- react
- axios
- flv.js
- bootstrap
- node-media-server
